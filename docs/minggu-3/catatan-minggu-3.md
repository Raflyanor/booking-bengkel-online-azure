# Catatan Minggu 3

Dokumentasi Minggu 3 berisi implementasi layanan web server pada Microsoft Azure Virtual Machine untuk proyek Sistem Booking Bengkel Online.

Implementasi yang dilakukan meliputi menyalakan VM, koneksi SSH, instalasi Nginx, pengujian akses melalui Public IP, pembuatan halaman web sederhana, pengujian curl localhost, dan penghentian VM setelah pengujian selesai.

Layanan web berhasil dijalankan melalui Public IP Address 104.214.169.211 menggunakan Nginx pada Ubuntu Server 24.04 LTS.
