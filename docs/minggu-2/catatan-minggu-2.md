# Catatan Minggu 2

Dokumentasi Minggu 2 berisi implementasi infrastruktur dasar pada Microsoft Azure untuk proyek Sistem Booking Bengkel Online.

Resource yang dibuat meliputi Resource Group, Virtual Network, Subnet, Network Security Group, Virtual Machine, Public IP Address, Network Interface, Disk, dan SSH Key.

Seluruh resource utama ditempatkan pada region East Asia.
