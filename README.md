# booking-bengkel-online-azure

Sistem Booking Bengkel Online Berbasis Cloud Microsoft Azure.

Repository ini digunakan untuk menyimpan dokumentasi, laporan mingguan, diagram arsitektur, screenshot konfigurasi cloud, dan file pendukung proyek Cloud Computing.

## Deskripsi Proyek

Sistem Booking Bengkel Online merupakan aplikasi berbasis web yang dirancang untuk mempermudah pelanggan dalam melakukan booking layanan servis kendaraan secara online. Sistem ini juga membantu admin bengkel dalam mengelola data pelanggan, layanan, jadwal servis, dan data booking.

## Platform Cloud

Proyek ini menggunakan platform Microsoft Azure dengan region utama:

**East Asia**

## Layanan Azure yang Digunakan

- Azure Virtual Machine
- Azure Virtual Network
- Network Security Group
- Public IP Address
- Azure Database for MySQL
- Azure Blob Storage
- Azure Monitor
- Microsoft Entra ID
